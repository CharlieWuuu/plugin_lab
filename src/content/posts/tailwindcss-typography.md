## 快速替 Markdown 排版！

如果開發的網站的內容是來自 markdown，想要為引入的文字套上樣式的話可以怎麼做？

可以用 `tailwindcss-typography` 來幫助你！

這個套件的用意是幫「乾淨的 HTML」套上一些預設樣式，讓他看起來好看一點。

但剛好 markdown 也是以基本的 HTML 為主嘛，

所以也可以用這個套件來美化。

我來簡述一下步驟：

- （以 markdown 為文章來源）
- 安裝 tailwind
- 設定 classname

```javascript
 <article className="prose">
 </article>
```
